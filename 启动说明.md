# AI古诗小老师 - 快捷启动说明

## 一键启动

双击运行：
```
一键启动全部.bat
```

位置：
```
F:\trae\tiku\poetry-learning\一键启动全部.bat
```

---

## 启动后访问

浏览器会自动打开，或手动访问：
```
http://localhost:8888/index_optimized.html
```

---

## 功能说明

1. **AI对话** - 点击麦克风说话，或使用快捷回复
2. **背诵挑战** - 点击"开始背诵"按钮
3. **学习报告** - 点击"📊 学习报告"按钮
4. **诗句详解** - 点击诗句卡片展开详情
5. **配置AI** - 点击"配置智能AI"切换DeepSeek模式

---

## 停止服务

关闭弹出的两个命令行窗口即可。
