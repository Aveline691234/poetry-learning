## 一、测试环境

### 1.1 测试环境要求

| 类型 | 要求 |
|------|------|
| 浏览器 | Chrome 90+ / Edge 90+ / Safari 14+ |
| 操作系统 | Windows 10+ / macOS 10.15+ / iOS 13+ / Android 8+ |
| 网络 | 4G/5G/WiFi（测试AI功能需要网络） |
| 硬件 | 麦克风（必须）、摄像头（可选） |

### 1.2 测试前置条件

1. **本地开发环境**
   ```bash
   # 启动后端服务
   cd poetry-learning
   python backend.py

   # 启动前端（使用浏览器直接打开HTML文件，或使用Live Server）
   ```

2. **Vercel部署环境**
   - Vercel项目已创建
   - DEEPSEEK_API_KEY已配置
   - 域名可访问
